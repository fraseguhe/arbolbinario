# Qué es un árbol binario.
>## Un árbol binario es una estructura de datos jerárquica y no lineal en la que cada nodo puede tener, como máximo, dos hijos, denominados "hijo izquierdo" e "hijo derecho". Se utiliza ampliamente para organizar datos de manera eficiente, permitiendo búsquedas rápidas al dividir la información en subárboles izquierdo (menores) y derecho (mayores) desde un nodo raíz. 
# Cómo se implementó.
>## ArbolBinario.java
>>### El árbol se implementó en 3 modulos:
>>### Nodo — guarda un número y dos referencias (izquierda y derecha)
>>### Arbol — contiene la raíz y todos los métodos
>>### main — crea el árbol, inserta los números y prueba los métodosElemento 1
>>### y con 3 metodos:
>>### añadir — baja recursivamente por el árbol hasta encontrar un espacio null
>>### buscar — recorre el árbol con un while comparando en cada nodo
>>### orden — visita izquierda → nodo → derecha, imprimiendo los números de menor a mayor
